# Give Hamster the full picture from Linear

Connect your Linear workspace so Hamster can reference your issues and projects.

## Linear Context Integration

Connect your Linear workspace and Hamster can reference your issues and projects when generating briefs or blueprints. If you're planning a feature in Linear, Hamster pulls in that context automatically. Connect Linear from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/linear).

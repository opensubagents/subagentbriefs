# Ship code from your briefs in parallel

Cloud Agents take a brief from plan to PR. Trigger delivery from chat or the Plan tab.

## Cloud Agents

Hamster can now ship working code from your briefs. Hand a brief off to a Cloud Agent and pick up the next one while it works.

![A brief chat thread where the user asks Hamster to deliver](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/chat-ask-light.gif?v=6)![A brief chat thread where the user asks Hamster to deliver](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/chat-ask-dark.gif?v=6) 

Track progress inline as Hamster works through the plan.

![Hamster's Cloud Agent activity streaming live as it works through the plan](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/agent-logs-light.gif?v=6)![Hamster's Cloud Agent activity streaming live as it works through the plan](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/agent-logs-dark.gif?v=6) 

Available on Team and Enterprise plans. Connect [GitHub](https://tryhamster.com/docs/hamster-studio/connections/github) from Settings > Connections and [learn more about Cloud Agents](https://tryhamster.com/docs/hamster-studio/cloud-agents).

## Plans in Chat

You can now generate plans directly from unified chat.

![A unified chat thread where the user asks Hamster to generate plans for two briefs and Hamster responds with two Plan Generating cards inline](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/plans-in-chat-light.gif?v=6)![A unified chat thread where the user asks Hamster to generate plans for two briefs and Hamster responds with two Plan Generating cards inline](https://tryhamster.com/_site/_site/images/releases/studio/1.2.0/plans-in-chat-dark.gif?v=6) 

[Learn more about chat](https://tryhamster.com/docs/hamster-studio/conversations).

## Improvements

* You can now link multiple briefs or initiatives at once from any list
* Parent threads now show when Hamster is waiting on your input from a branch
* Added Routines to mobile navigation

## Bug Fixes

* Fixed code blocks, tables, and images showing up broken in Hamster's replies
* Fixed Hamster's chat suggestion cards getting stuck mid-run
* Fixed form submissions failing after a page had been open for a while

# Create briefs, blueprints, and methods directly from the terminal

Your AI coding agent can now set up an entire project in Hamster from a single conversation.

## CLI Create Commands

You can now run `hamster brief create`, `hamster blueprint create`, and `hamster method create`. Describe what you're building and your agent sets up the briefs, blueprints, and methods Hamster needs to start planning.

![An AI coding agent using the Hamster CLI to create a brief, blueprint, and method, then showing a summary table with all three synced to the team](https://tryhamster.com/_site/images/releases/studio/0.46.0/cli.webp)

[Learn more about the Hamster CLI](https://tryhamster.com/docs/hamster-studio/cli).

## Bug Fixes

* Task commands now accept any team prefix (e.g., `HOR-133`, `DEV-42`), not just `HAM-`
* CLI auth now works reliably with environment tokens on remote servers and containers

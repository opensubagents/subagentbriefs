# Teach Hamster your team's playbook

Author reusable skills that shape how Hamster approaches your team's work.

## Skills

Every team has knowledge that lives in people's heads: how to write a brief for enterprise clients, which API patterns to follow, what the brand voice sounds like. You can now capture that as skills, reusable instructions that Hamster picks up automatically.

![The Skills hub showing skill cards like Cycle Retro Facilitation and Merchant Release Changelog organized by category](https://tryhamster.com/_site/images/releases/studio/0.51.0/cover.webp)

Attach skills to your methods. Hamster selects the relevant ones automatically.

![The Skills tab on a User Story Mapping method showing available skills to add, including Cycle Retro Facilitation, Merchant Release Changelog, and Onboarding Funnel Analysis](https://tryhamster.com/_site/images/releases/studio/0.51.0/method-skills.webp)

Import bundled skills, write your own in the built-in editor, or import them from a GitHub repository. [Get started with skills](https://tryhamster.com/docs/hamster-studio/skills-methods/skills-overview).

## Slack Feedback

Submit product feedback directly from Slack with `/hamster feedback`. Describe what you'd like to see changed, and Hamster creates a feedback issue with your context attached. No need to leave your conversation.

## Figma URL Context

Paste a Figma URL into chat, and Hamster extracts the design and adds it to your brief's context. Thumbnails appear in the Context tab.

Private files use your connected Figma account; public files work without one. Connect Figma from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/figma).

## Improvements

* Linear sync now supports filtering by tag to control which projects are synced
* Context search retries with a wider scope when the first pass returns nothing across your [connected services](https://tryhamster.com/docs/hamster-studio/connections/overview)
* Thread summaries now live in a collapsible bar at the bottom of the thread instead of a floating button
* Hamster's thinking steps now show formatted text instead of unformatted markdown
* New Google Drive connections request only read access (existing connections are unaffected)
* The Slack bot now shows progressive status updates while processing your message instead of a static indicator
* Connecting tools (Slack, Figma, Jira, Google Drive, Notion) now opens a popup instead of navigating away from the page
* The AI assistant can no longer be accidentally removed from a team — protected membership prevents deletion
* Skills can be synced between your code repository and Hamster via the [CLI](https://tryhamster.com/product/studio/cli) with `hamster skills sync`

## Bug Fixes

* Fixed synced Linear sub-issues not appearing under the correct parent task
* Fixed an error when switching a Linear connection from two-way sync to search-only
* Fixed synced Linear tasks arriving with missing details
* Linear sync no longer creates tasks for connections that have been disabled
* Fixed invite notifications being sent to all account members instead of just the relevant team
* Fixed pending questions disappearing on page refresh

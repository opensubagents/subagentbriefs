# Stay in the loop on every decision

Hamster now pauses mid-workflow to ask for clarification or get your approval before proceeding.

## Hamster Asks Before It Acts

Hamster can now pause to ask for clarification or confirmation. When researching a topic, it might ask which sources you prefer before proceeding, or request approval before adding findings to a brief.

If Hamster needs to know your company's target market before writing a brief, it asks rather than guessing. [Learn more about conversations](https://tryhamster.com/docs/hamster-studio/conversations).

## Improvements

* After onboarding, Hamster automatically generates an "About Us" blueprint tailored to your company type
* Complex requests are now broken into individual tasks and handled automatically

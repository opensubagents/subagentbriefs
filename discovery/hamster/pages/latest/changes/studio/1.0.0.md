# Let Hamster pick up work the moment something changes

Routines run Hamster on brief, task, initiative, and blueprint events. Results post into the relevant chat.

## Routines

Hamster can now draft a summary the moment a brief is marked blocked, or post a standup update when tasks complete.

![The Routines list page showing ten routines paired with their workspace event triggers and last run status](https://tryhamster.com/_site/_site/images/releases/studio/1.0.0/cover-light.webp?v=6)![The Routines list page showing ten routines paired with their workspace event triggers and last run status](https://tryhamster.com/_site/_site/images/releases/studio/1.0.0/cover-dark.webp?v=6) 

Each routine has a trigger and an instruction.

![The 'Surface related work' routine detail view with name, instructions, a When Brief status is created trigger, and connector pills for GitHub, Linear, Google Drive, and Slack Search](https://tryhamster.com/_site/_site/images/releases/studio/1.0.0/routine-detail-light.webp?v=6)![The 'Surface related work' routine detail view with name, instructions, a When Brief status is created trigger, and connector pills for GitHub, Linear, Google Drive, and Slack Search](https://tryhamster.com/_site/_site/images/releases/studio/1.0.0/routine-detail-dark.webp?v=6) 

Hamster posts results into the relevant brief or blueprint chat, or into your notifications when a routine proposes a document edit.

[Learn more about Routines](https://tryhamster.com/docs/hamster-studio/routines).

## Faster Workspace Search

Chat answers, plan generation, and brief drafts now pull context in milliseconds. Searches that used to hang for up to a minute return instantly.

## Improvements

* Brief prompts now ask about dependencies and blockers instead of timelines
* Hamster now uses your uploaded chat attachments when creating briefs and blueprints
* Emoji now look the same in chat and documents on every device

## Bug Fixes

* Fixed Linear sync flipping brief status back to draft on unrelated updates
* Fixed Jira sync silently dropping briefs that exceed Jira's size limit
* Fixed failed blueprint deletions reporting as successful
* Fixed Hamster saying "I cannot create a new brief" then creating one anyway
* Fixed newly created briefs opening as empty documents
* Fixed brief search returning every result as "Untitled" when searching by name
* Fixed initiative search in link dialogs returning incomplete results
* Fixed sign-out from the account selector leaving you signed in
* Fixed the sign-in page briefly flashing on load
* Fixed form submissions failing for some users

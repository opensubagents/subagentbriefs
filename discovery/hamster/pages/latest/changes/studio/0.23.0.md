# Find any conversation instantly

Chat threads auto-generate titles based on conversation content.

## Chat History and Thread Titles

Chat threads automatically generate titles based on the conversation content. Instead of scrolling through untitled threads, you can spot the one you need at a glance. [Learn more about conversations](https://tryhamster.com/docs/hamster-studio/conversations).

## Focused Context

When you ask a question, Hamster only includes relevant context. If you're asking about a specific product feature, it won't pull in unrelated company information.

* User messages are easier to read with a lighter background

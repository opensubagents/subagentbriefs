# Your tasks, always in sync across all your tools

Two-way sync with Linear, plus guided onboarding that adapts to your workflow.

## Linear Two-Way Sync

We've added two-way sync with Linear. Connect your workspace, and status changes flow both ways automatically.

![The Linear connection dialog showing permissions and two-way sync toggle, alongside the connected card with a Synced status badge](https://tryhamster.com/_site/images/releases/studio/0.43.0/linear-connect.webp)

Tasks created from your plan link directly to Linear issues. Move a task in either tool and the other updates.

![The Plan tab showing synced tasks with both Hamster and Linear issue IDs visible on each row](https://tryhamster.com/_site/images/releases/studio/0.43.0/linear-tasks.webp)

Rolling out to early access customers in the coming weeks. Connect Linear from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/linear).

## Guided Onboarding

Hamster now walks new users through setup based on what they're trying to do. Setting up a content pipeline? It focuses on briefs and blueprints. Researching a topic? It pulls in the context tools first.

## Improvements

* You can now tell when Hamster is working on a reply instead of wondering if it stalled
* Run `hamster update` to check for and install the latest CLI version

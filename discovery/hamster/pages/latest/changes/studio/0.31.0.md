# Hamster can research topics across the web

Ask Hamster to look into a competitor or industry trend and it returns structured findings for your briefs.

## Web Research

Hamster can now research topics on the web and bring back results to inform briefs and conversations. If you need to understand a competitor's positioning or an industry trend, Hamster works through sources and returns structured findings. [Learn more about guided research](https://tryhamster.com/docs/hamster-studio/plans/guided-research).

## Improvements

* Chat messages now show timestamps, helpful for async teams working across time zones
* Paste or drag-and-drop images directly into the document editor
* Briefs, Blueprints, and Methods show context-specific suggestions when empty
* Thread titles are now clickable in the reply indicator for easier navigation

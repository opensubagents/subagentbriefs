# Connect Google Drive so Hamster can reference your docs

Link your Google Drive account and Hamster pulls in your documents when generating briefs and blueprints.

## Google Drive Connector

Connect your Google Drive account and Hamster can reference your documents when generating briefs, blueprints, and other content. Connect Google Drive from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/google-drive).

## Shared Thread Summaries

When a thread is shared, people now see a summary preview before diving into the full conversation.

## Improvements

* The GitHub connection dialog now links directly to Personal Access Token creation with guidance on which permissions to select

## Bug Fixes

* Fixed document headings not displaying correctly throughout the app
* Fixed the block-type dropdown getting clipped behind other editor elements
* Fixed threads pulling in unrelated context from other conversations
* Fixed pasted URLs not retrying reliably when processing fails
* Fixed collaborative editing cursors not identifying each participant correctly

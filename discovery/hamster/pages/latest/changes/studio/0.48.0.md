# A clearer picture of how your briefs evolve

The redesigned Activity tab groups votes, status changes, and comments by document version.

## Activity Tab Redesign

The Activity tab now groups everything by document version. Expand any version to see what happened: who voted, what changed, and what the team discussed.

![The current version section expanded, showing the blue dot indicator, brief card with Continue Editing shortcut, and status change badges](https://tryhamster.com/_site/images/releases/studio/0.48.0/current-version.webp)

If someone raised a concern on Version 3 and the team resolved it by Version 4, the whole arc is visible in one scroll.

![The full Activity timeline showing older versions with status change badges, alignment votes, and a Show older versions link](https://tryhamster.com/_site/images/releases/studio/0.48.0/timeline.webp)

## @Mention Notifications

@mention someone in a thread and they're added as a participant. They see the full thread history, not just the latest message.

## Auto-Threading

When Hamster's response is detailed enough to need its own space, it automatically moves the conversation into a dedicated thread.

## Improvements

* Attach `.fig` design files to a [brief's](https://tryhamster.com/product/studio/briefs) context or drop them into chat. Hamster reads the design to inform its output
* Brief list preferences (filters, sort, selected team) now persist across sessions

## Bug Fixes

* Fixed an issue where documents could be deleted when the user who last updated them was removed from the team

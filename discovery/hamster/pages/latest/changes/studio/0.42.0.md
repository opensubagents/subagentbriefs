# Your coding agents, wired into the plan

The CLI and MCP server give your coding agents live access to briefs, tasks, and plans. Plus document version history and six new languages.

## CLI: Real-Time Context for Your Coding Agent

The CLI syncs your briefs, tasks, blueprints, and methods into your repo as plain markdown. Your coding agent always has current requirements without anyone copying anything.

A built-in terminal dashboard shows your briefs and tasks in a kanban layout you can drill into without leaving the terminal. [Get started with the CLI](https://github.com/gethamster/cli), or [learn more about CLI sync](https://tryhamster.com/docs/hamster-studio/cli/cli-sync).

## MCP Server

The MCP server gives your coding agent tools to browse briefs, pick up tasks, create subtasks, and mark work done from inside the coding session. When a task is finished, the plan in Studio updates automatically.

Works with Claude Code, Codex, Cursor, Windsurf, Gemini CLI, and any MCP-compatible tool. [See all available tools](https://tryhamster.com/docs/hamster-studio/mcp/available-tools).

## Document Version History

Every document now tracks its full version history. Browse every saved state, compare any two versions with a structural diff, and restore previous content in one click.

Save named checkpoints at meaningful moments. AI-generated summaries on each version help you find the right state at a glance.

## Translations

Hamster Studio is now available in six new languages: Chinese, Japanese, Spanish, French, German, and Portuguese.

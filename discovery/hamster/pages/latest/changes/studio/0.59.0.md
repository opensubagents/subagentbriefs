# Bring your designs into the conversation

Connect a Figma workspace and Hamster can reference your pages and frames in briefs, plans, and chat.

## Figma connector

Hamster can now reference your Figma pages and frames when generating briefs, writing documents, or answering questions in chat.

![The "Connect Figma to Hamster" dialog describing how Hamster uses Figma for shared context, with permissions to read file content, components, styles, and dev resources, and a Connect Figma button](https://tryhamster.com/_site/images/releases/studio/0.59.0/figma-connect.webp)

Connect from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/figma) and choose which files to include.

## Improvements

* `hamster auth login` now works on systems without a keychain, like WSL and remote Linux
* Hamster can now answer from pinned brief versions
* Hamster now pulls from review requests and approvals when answering questions
* The deliver dialog branch selector no longer caps at 100 branches
* Delivery PRs now open as drafts and show who triggered the delivery

## Bug Fixes

* Fixed duplicated title headings in initiatives generated by Hamster
* Fixed the initiative editor overflowing into the briefs and sub-initiatives sections
* Fixed archived briefs appearing in the "Add Brief" dialog for initiatives
* Fixed thread chat comments incorrectly appearing in brief activity timelines
* Fixed Slack connector losing search access after reconnecting
* Fixed intermittent Linear task updates not syncing

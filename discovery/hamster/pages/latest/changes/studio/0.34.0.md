# Share integration URLs and Hamster pulls the real data

Paste a Google Drive, Notion, Jira, or GitHub URL in chat and Hamster queries your connected integration directly.

## Integration-Aware URLs

Paste a URL from a connected integration in chat and Hamster pulls the real data directly. Paste a Jira ticket URL and Hamster fetches the full ticket details from your Jira connection instead of scraping the page.

Supported integrations: Google Drive, Notion, Jira, and GitHub. Set up your connections from [Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/overview).

# Ask Hamster about the files you've uploaded

Every PDF, image, and text document is now searchable context, plus Blocked and Archived initiative states and Linear sync for sub-initiatives.

## Files in the Context Graph

Uploaded files used to be stored but not searchable. Every PDF, image, and text document you upload is now searchable context, alongside the documents you write in the editor.

![The Referenced Context panel listing a URL still processing, an SVG diagram still processing, a PDF roadmap, and a text interview file, above a chat where Hamster answers a question about acceptance criteria by citing the Shopify Plus Integration Spec and Fulfillment SLA Requirements from that uploaded context](https://tryhamster.com/_site/images/releases/studio/0.58.0/files-context.webp)

Hamster cites the specific passage from an uploaded file when that file answers the question.

## Blocked and Archived states

Initiatives can now be marked **Blocked** or **Archived**.

![The initiatives list with Owner, Target, and State columns, with the State picker open on a Shipping initiative and showing Draft, Refining, Shipping, Done, and the new Blocked and Archived options](https://tryhamster.com/_site/images/releases/studio/0.58.0/initiative-status.webp)

## Improvements

* Hamster now cites sources more precisely when answering from context
* Nested initiatives now also sync with Linear. [Connect from Settings > Connections](https://tryhamster.com/docs/hamster-studio/connections/linear)
* Re-syncing URL documents is now limited to members with document management permissions

## Bug Fixes

* Fixed live updates dropping during collaboration
* Fixed initiatives failing to create or save edits
* Fixed the "Create Pull Request" button not appearing after a branch push
* Fixed vote outcomes showing wrong details in activity timelines
* Fixed intermittent login failures

# The right questions, asked upfront

Structured discovery questions let Hamster collect the right information step by step.

## Interactive Discovery Questions

Hamster can now pause mid-conversation to ask structured questions before continuing. Each question appears as selectable options you can click through, or type a custom answer when none of the options fit.

## Skills and Methods Redesign

Skills and methods now live on dedicated pages with their own navigation. A new template library lets you browse and preview templates before activating them. [Learn more about skills and methods](https://tryhamster.com/docs/hamster-studio/skills-methods).

## Automatic Sub-Threads

When a request involves multiple pieces of work, such as analyzing several uploaded documents, Hamster creates a dedicated thread for each piece and combines the results.

## Improvements

* Web research now delivers citation-backed results and runs multiple queries in parallel
* Briefs get an automatic title as soon as Hamster writes the first content
* Blueprint and method sidebars update in real time as Hamster creates or modifies documents

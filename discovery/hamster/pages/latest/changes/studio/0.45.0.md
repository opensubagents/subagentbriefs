# See exactly what changed, word by word

Inline diffs now highlight only the changed words, not entire paragraphs.

## Inline Diffs

Reviewing edits used to mean scanning the entire document for green highlights. Hamster now shows clean inline diffs with only the changed words marked. No more walls of green.

![Inline diff showing word-level changes in a document with strikethrough removals, highlighted additions, and Reject/Accept controls](https://tryhamster.com/_site/images/releases/studio/0.45.0/inline-diff.webp)

## Improvements

* New users get a guided introduction when they first open the Plan tab
* View full task and brief details from the CLI with `hamster task get <id>` and `hamster brief get <slug>`

# Hamster remembers what you said earlier in the conversation

Chat history is now used as context when Hamster processes your requests.

## Conversational Context

Hamster now references your earlier messages when processing requests. If you discussed your target audience three messages ago and then ask Hamster to write a brief, it factors in that context automatically. [Learn more about conversation history](https://tryhamster.com/docs/hamster-studio/conversations/conversation-history).

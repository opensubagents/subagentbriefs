# Humans + Agents

AI agents are joining your team. The question isn't whether you'll work with them — it's how you'll structure the collaboration.

The most productive teams aren't just using AI to write code faster. They're rethinking the entire workflow: which decisions need human judgment, which tasks can be delegated to agents, and how to maintain quality as execution accelerates.

**Humans make decisions. Agents execute them.** Hamster is built for this model. You write the brief, align the team, and approve the plan. AI generates the tasks and helps execute them.

The result is a team that ships faster without losing control. Human judgment stays in the loop. Agent execution scales the output.

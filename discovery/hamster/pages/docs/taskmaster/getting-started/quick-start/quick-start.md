# Quick Start

This guide is for new users who want to start using Taskmaster with minimal setup time.

It covers:

* [Requirements](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/requirements): You will need Node.js and an AI model API Key.
* [Installation](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/installation): How to Install Taskmaster.
* [Configuration](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/configuration-quick): Setting up your API Key, MCP, and more.
* [PRD](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/prd-quick): Writing and parsing your first PRD.
* [Task Setup](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/tasks-quick): Preparing your tasks for execution.
* [Executing Tasks](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/execute-quick): Using Taskmaster to execute tasks.
* [Rules & Context](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/rules-quick): Learn how and why to build context in your project over time.

> **By the end of this guide, you'll have everything you need to begin working with Taskmaster.**

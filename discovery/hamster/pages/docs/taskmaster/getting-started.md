# Getting Started

Welcome to Taskmaster! This section will guide you through everything you need to know to get up and running quickly.

## Quick Start Guide

For new users who want to start using Taskmaster with minimal setup time, we recommend following our comprehensive Quick Start guide:

[Start the Quick Start Guide →](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/quick-start)

This guide covers:

* **Requirements**: Node.js and AI model API keys
* **Installation**: How to install Taskmaster
* **Configuration**: Setting up API keys, MCP, and more
* **PRD Creation**: Writing and parsing your first PRD
* **Task Setup**: Preparing your tasks for execution
* **Executing Tasks**: Using Taskmaster to execute tasks
* **Rules & Context**: Building context in your project

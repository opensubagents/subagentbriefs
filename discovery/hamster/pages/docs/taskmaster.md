# Introduction

Taskmaster is your AI's personal project manager. Organize, research, expand, prioritize, and ship tasks effortlessly. Enjoy permanent context, zero drift & instant clarity. It's free, open-source & API-friendly. Let's vibe productively.

## Quick Install

`npm install -g task-master-ai` 

Then run:

`task-master init` 

## [Getting Started](https://tryhamster.com/docs/taskmaster/getting-started/quick-start/quick-start)

Designed for first-time users. Get set up, create your first PRD, initialize your project, and run your first task.

## [Task Workflow](https://tryhamster.com/docs/taskmaster/task-workflow/tags)

Organize work with tags, manage dependencies, research live information, and adjust task scope. The day-to-day features that keep your project moving.

## [Automation](https://tryhamster.com/docs/taskmaster/automation/start)

Launch tasks with full context using `start`, or run automated loops that pick up tasks, implement them, and move to the next one.

## [AI Providers](https://tryhamster.com/docs/taskmaster/ai-providers/overview)

Works with 15+ AI providers including Anthropic, OpenAI, Google, Perplexity, and local models like Ollama and LM Studio. Bring your own keys.

## [IDE & Editor Setup](https://tryhamster.com/docs/taskmaster/ide-setup/supported-editors)

Integrates with 13+ editors including Cursor, Claude Code, VS Code, Windsurf, Zed, and more. One command to configure.

## [Team Collaboration](https://tryhamster.com/docs/taskmaster/team/overview)

Go from solo development to team collaboration with cloud storage, real-time sync, and shared task management via Hamster.

## [Best Practices](https://tryhamster.com/docs/taskmaster/best-practices/index)

Advanced workflows, configuration tips, and real-world usage patterns. Recommended for active users.

## [Technical Capabilities](https://tryhamster.com/docs/taskmaster/capabilities/mcp)

Complete reference for CLI commands, MCP tools, task structure, and cluster execution.

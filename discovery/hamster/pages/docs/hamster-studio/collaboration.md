# Collaboration

Work together on documents and [Briefs](https://tryhamster.com/docs/hamster-studio/briefs) in real time — with live editing, mentions, and full version history.

## Overview

Hamster Studio is built for teams working together on documents, [Briefs](https://tryhamster.com/docs/hamster-studio/briefs), and plans. Collaboration features let multiple people work on the same content at the same time, stay informed when something needs their attention, and recover any version of a document when things change direction.

## In This Section

* [Real-Time Editing](https://tryhamster.com/docs/hamster-studio/collaboration/real-time-editing) — Multiple team members can edit [Briefs](https://tryhamster.com/docs/hamster-studio/briefs) and documents simultaneously. Cursor positions and edits from each person appear live, with no need to manually merge changes.
* [Mentions and Notifications](https://tryhamster.com/docs/hamster-studio/collaboration/mentions-notifications) — @mention teammates directly inside documents or conversations. Notifications keep everyone informed when they're mentioned or when something relevant changes.
* [Document Versioning](https://tryhamster.com/docs/hamster-studio/collaboration/document-versioning) — Every edit to a [Brief](https://tryhamster.com/docs/hamster-studio/briefs), skill, or [Method](https://tryhamster.com/docs/hamster-studio/skills-methods) is tracked automatically. You can view the full history of a document, save named checkpoints, and restore any prior version.
